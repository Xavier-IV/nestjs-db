## Description

This repository is to complement my Youtube video on NestJS Database series.

[Youtube Video](https://youtu.be/fBvQL176fHg)

## Installation

Be sure to have docker install.

```bash
$ npm install
$ docker-compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
